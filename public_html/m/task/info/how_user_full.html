1) перейдите по ссылке в меню Заказать, укажите предмет стоимость, после загрузки задания появится номер заказа, запомните его.

2) периодически посещайте страницу проверки статуса заказа, чтобы узнать о его готовности. Как только в ответ на запрос появится ссылка 'Выбери решающего', перейдите по ней и вы увидите что наши Авторы предлагают решить задание по разным ценам. Вы можете выбрать решающего, который готов выполнить задание за меньшую сумму. На момент выбора на вашем счету должно быть 50% суммы заказа, если это не так, будет предложено пополнить баланс. После выбора Автора решение вашего задания начинается.

3) периодически проверяйте статус задания. Как только заказ будет готов, при проверке появится ссылка для скачивания решения. Перейдите по ней.

4) перед скачиванием, система попросит вас оплатить оставшиеся 50% суммы заказа одним из удобных для вас способов. После оплаты автоматически начнется загрузка решения.
